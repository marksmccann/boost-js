jquery-plugin-maker

This plugin establishes a consist pattern for creating jquery applications and adds core functionality to jump-start the development process.

view source download source

Installing

Include on html page via script tags

<script src="https://code.jquery.com/jquery-3.1.0.min.js"></script>
<script src="https://cdn.rawgit.com/marksmccann/plugin.js/master/plugin.js"></script>

install via npm

$ npm install jquery-plugin-maker

creating a plugin

$.plugin.add( '', appObject, defaultSettings )

reserved attributes

<span id="" data-init=""></span>
<span data-bind="" data-role=""></span>

settings

<span id="" data-init="" data-[whatever]=""></span>

instantiating a plugin

$('your-element').appName();
